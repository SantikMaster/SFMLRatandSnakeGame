# SFMLRatandSnakeGame
SFML new game and menu

Game with SFML and AStar pathfinding algorithm. Imagine you are a rat, you can dig holes and make a dirty hills.
Collect potatoes and avoid the snake.


#Installation
1)instal SFML
2)Set path to SFML in cmakeLists

then 
$ mkdir build
$ cd build
$ cmake ..
